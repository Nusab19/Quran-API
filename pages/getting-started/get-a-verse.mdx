import { Callout } from "nextra/components";

## Request Format

Make a **GET** request to `/<surahNo>/<ayahNo>.json`

<Callout type="info">Replace `<surahNo>` with the Surah Number and `<ayahNo>` with the Ayah Number</Callout>

## Response

A _JSON_ object containing information about the Ayah.
